SMC4PEP converter


SMC4PEP is a tool for converting a business process, expressed in the BPMN format, into an equivalent Markov Decision Process (MDP) expressed in the language of the PRISM model checker. The generated MDP file can then be used for model checking the process and verifying its properties. To learn more about it, click here.
